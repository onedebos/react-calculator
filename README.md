## Calculator

This is a calculator app built using React.

## Built With

- React
- npm
- Webpack

## Linters used
- ESlint


## Live Demo

[Live Demo Link](https://onedebos.github.io/weather-app/)

## Clone the project
```
git clone https://github.com/onedebos/weather-app.git
```

## Install dependencies

```
npm install 
```
or

```
yarn install 
```

## Run the app
```
yarn start
```

or

```
npm start 
```
