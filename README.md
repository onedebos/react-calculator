## Calculator

This is a calculator app built using React.

## Built With

- React
- npm
- Webpack

## Linters used
- ESlint


## Live Demo

[Production version](https://debs-calculator-2.herokuapp.com/)

## Clone the project
```
git clone https://github.com/onedebos/calculator.git
```

## Install dependencies

```
npm install 
```
or

```
yarn install 
```

## Run the app
```
yarn start
```

or

```
npm start 
```
